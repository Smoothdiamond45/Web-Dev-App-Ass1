Frame & Light
A photography portfolio web application built to showcase collections across multiple genres. Browse curated photo collections with detailed camera metadata for each shot.

About
Frame & Light is a photography portfolio site built as a college assignment. It displays personal photography across five collections — Street, Portrait, Landscape, Macro, and Animals — with a clean, dark aesthetic designed to let the images speak for themselves.

Pages

Home — Landing page introducing the portfolio
Dashboard — Overview of all collections
Collection Pages — Individual pages for each photography category (Street, Portrait, Landscape, Macro, Animals)
About — Background on the photographer


Features

Browse photos across multiple collections
Each photo displays camera metadata (aperture, shutter speed, ISO, lens)
Smooth hover animations on photo cards
Responsive grid layout

Frontend  -- HTML, CSS, JavaScript
Templating --Handlebars (.hbs) 
Backend --Node.js
Deployment --Render
